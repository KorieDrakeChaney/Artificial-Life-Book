class Grid {

class Grid {

Creating The Cell Class

Now for this one, let’s create a basic Cell class. This class holds a state, id and position([x, y]) which will be changed through the passing arguments.


class Cell { 
    constructor(state, id, position){
        this.state  = state // 0 or 1
        this.id = id // 0.....infinity
        this.position = position // [x, y] | [0, 0]


        // logic
    }
}

Now within this constructor, let’s create the graphics using the document api, which will allow us to create many features.

The function we will be using is document.createElement(…). This will return whatever Element pertaining to the argument we give it:


    //examples

let div = document.createElement('div'); // returns a div;
let span = document.createElement('span'); // returns a span;
let h1 = document.createElement('h1'); // returns a h1;

let unknown = document.createElement('aslkdjasjlkd'); // return an HTMLUnknownElement.

This will allow us to create the cell’s body, in this case, a div.


class Cell { 
    constructor(state, id, position){
        // earlier stuff

        this.body = document.createElement('div');
    }
}

Now we have the body, again, it’s doing nothing, but we will start adding to it, like the color. To change the color, we have to learn about the HTMLElement, like the div we have above. If you read through the documentation, you can find that we can grab the style of the element and change it. Meaning, we can change it’s backgroundColor!

Good tip : Always read through documentation! 

    //earlier stuff

    this.body.style.backgroundColor = 'red' // whatever, your choice.

In the ’ ’ you can put any valid color


        //examples

    this.body.style.backgroundColor = 'red' // returns red;
    this.body.style.backgroundColor = 'rgb(255, 0, 0)' // returns red, again;
    this.body.style.backgroundColor = 'rgba(255, 0, 0, 0.5)' // returns red, but half opacity!;
    this.body.style.backgroundColor = '#ff0000' // returns red, again again;
    this.body.style.backgroundColor = 'hsl(0, 100%, 50%)' // returns red, again again again;

Now, we have changed the cells color. How can we see it ? Well for one, we haven’t even added it to the document yet, in order to this, we must use the Node.appendChild(…) function. The Node that we will be appending our cell to, is the body, in order to retrieve the body(not to be confused with the cell’s body), we must go through the document:


class Cell { 
    constructor(state, id, position){
        // earlier stuff

        this.body = document.createElement('div');
        this.body.style.backgroundColor = 'red'
        document.body.appendChild(this.body); // now whenever we initialize the cell, we should see it !
    }
}

Using Cell Class in Grid Class

Now let’s create the cells! Remember the nested for loops we had earlier? Let’s go back into the Grid class , and add some new lines of code.

class Grid {
    constructor(width , height){
        //earlier stuff
        for(let column = 0; column < this.width; column++){
            for(let row = 0; row < this.height; row++){
                let cell = new Cell(0, 0, [column, row]);
            }
        }
    }
}

let’s see what we get !

Wait… A BLANK SCREEN ? Well, the reason for this is that the cell’s body, has no width and height

For this problem, I want to add a new variable within the Grid class, that will be passed into the cell. The new variable will be called cellSize, which we can have a default set to 20(px).


    constructor(width , height){
        //earlier stuff
        this.cellSize = 20;

        // for loops and stuff
    }

Now in the Cell class, we will add a the new variable and argument:


class Cell { 
    constructor(state, id, position, size){
        this.state  = state // 0 or 1
        this.id = id // 0.....infinity
        this.position = position // [x, y] | [0, 0]
        this.size = size; // 0.....infinity

        
    }
}

Then we set the change the width and height of the body, using again, the style.

class Cell { 
    constructor(state, id, position, size){
        // earlier stuff

        this.body.style.width = `${this.size}px`
        this.body.style.height = `${this.size}px`
    }
}

Now, if we did this right, we should have red squares ! Next chapter will be how to set up more of the grid logic : D.
